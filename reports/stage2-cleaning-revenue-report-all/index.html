    視覺化說明¶

    績效圖標與顏色說明

    • 🚀 超強成長 (30%+): 營收爆發性成長
    • 📈 強勢成長 (15-30%): 營收強勁增長
    • ✅ 穩健成長 (5-15%): 營收穩定成長
    • 🟢 微幅成長 (0-5%): 小幅正成長
    • ⚪ 零成長 (0%): 營收持平
    • 🟡 輕微下滑 (-10-0%): 短期調整
    • 🟠 明顯下滑 (-20--10%): 營收下降
    • 🔴 大幅下滑 (-20%+): 營收大幅衰退

    正成長區間 (淺→深表示關注度增加): 淺紅(50%+) → 番茄紅 → 橘紅 → 深紅 → 磚紅 → 暗紅(0-10%)

    零成長: 黑色(0%)

    負成長區間 (深→淺表示風險度降低): 最深綠(0至-10%) → 很深綠 → 深綠 → 森林綠 → 萊姆綠 → 淺綠(-50%+)

    使用說明

    • 總覽報告: 本檔案提供所有股票概況
    • 個別報告: 每檔股票都有獨立的詳細報告 (含EPS和P/E數據)
    • 資料更新: 報告根據最新可用資料產生
    • stage2-cleaning-revenue-report-all.md - 本總覽檔案
    • stage2-cleaning-revenue-report/stage2-cleaning-revenue-report-[股票代號].md - 各股票詳細報告
    • 本報告僅供參考，不構成投資建議
    • 請結合其他財務指標進行綜合分析
    • 注意產業景氣循環對營收的影響
    • 新增EPS和P/E倍數提供估值參考
