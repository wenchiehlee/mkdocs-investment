### :gear: 預測方法

本預測採用 **TTM + FactSet 動態整合模型**：

#### 1️⃣ TTM 方法 (Trailing Twelve Months)

```
TTM EPS = 最近4季 EPS 總和
TTM 股利 = TTM EPS × 平均配息率(5年) × 現金比例(5年)
```

#### 2️⃣ FactSet 時間加權方法

```
時間權重 = exp(-0.1 × 天數差距/30)  # 越近期權重越高
時間加權EPS = Σ(EPS × 時間權重) / Σ(時間權重)
FactSet 股利 = 時間加權EPS × 平均配息率(5年) × 現金比例(5年)
```
