### 如何選擇合適的投資期間？

!!! tip "投資期間選擇建議"

    **短期投資 (3個月 - 1年)**

    - 適合對象：積極交易者、波段操作投資人
    - 特點：較高波動性、需要頻繁監控
    - 策略：技術分析 + 估值判斷，把握短期錯殺機會
    - 風險：市場情緒影響大，需要較強風險承受力

    **中期投資 (2年 - 5年)**

    - 適合對象：成長股投資者、產業研究者
    - 特點：平衡成長性與穩定性
    - 策略：產業趨勢 + 公司基本面 + 估值水準
    - 風險：產業景氣循環影響，需注意產業變化

    **長期投資 (6年 - 10年)**

    - 適合對象：價值投資者、退休規劃者
    - 特點：穩健增長、複利效應顯著
    - 策略：優質企業 + 合理估值 + 長期持有
    - 風險：最低，但需要耐心與紀律

### 總報酬 vs 價格報酬

!!! note "報酬率計算方式"
    本系列報告採用**總報酬 (Total Return)** 計算方式：

    - ✅ **包含**: 股價變動 + 現金股利
    - ✅ **優點**: 反映真實投資報酬，尤其重要於高股息股票
    - ✅ **適用**: 長期投資分析、退休規劃、價值投資

    總報酬 = (期末價格 + 累計現金股利 - 期初價格) / 期初價格

### R²值解讀

!!! info "模型可靠度指標"
    R²值衡量本益比對未來報酬率的解釋能力：

    - **R² ≥ 0.7**: 強相關，本益比是主要決定因素
    - **0.4 ≤ R² < 0.7**: 中度相關，本益比有顯著影響
    - **R² < 0.4**: 弱相關，其他因素影響更大

    R²越高，預測越可靠；R²低的股票需特別注意其他影響因素。
